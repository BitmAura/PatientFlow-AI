# WhatsApp Business Setup - Simplified Guide

## 🚀 Quick Setup (2 Minutes)

### Step 1: Get Your WhatsApp Business API Credentials

**Option A: Use Our Recommended Provider (Easiest)**
1. Click "Connect WhatsApp" in your dashboard
2. We'll automatically create your WhatsApp Business account
3. No technical setup required!

**Option B: Use Your Own Meta Account (Advanced)**
1. Go to [Meta Developers](https://developers.facebook.com/)
2. Create a new app
3. Add WhatsApp Business API
4. Get your credentials below

### Step 2: Enter Your Credentials

Copy these values from your WhatsApp Business dashboard:

```
WhatsApp Phone Number ID:   _______________
WhatsApp Access Token:      _______________
Webhook Secret:            _______________ (optional)
```

### Step 3: Configure Webhook (Optional)

**For Advanced Users Only**

If you want to receive message status updates, set this webhook URL in your Meta dashboard:

```
Webhook URL: https://your-domain.com/api/webhooks/whatsapp
Verify Token: your-webhook-secret
```

## 🔧 Troubleshooting

**"Connection Failed"**
- Check your phone number ID is correct
- Ensure your access token hasn't expired
- Try refreshing the page and reconnecting

**"Messages Not Sending"**
- Check if you have available message credits
- Verify patient phone numbers are valid
- Ensure patients have opted in to WhatsApp messages

**"Webhook Errors"**
- This is optional - your messages will still work without webhooks
- Check your domain is accessible from the internet
- Verify your webhook secret matches
