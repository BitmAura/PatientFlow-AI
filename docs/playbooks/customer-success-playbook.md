# 📈 Customer Success Playbook (Clinic Onboarding)
**Version:** 1.0 (Audit Rectification)
**Goal:** Maximize Trial-to-Paid conversion and prevent "Day 3 Abandonment".

## Day 0: The Setup Phase
- [ ] **Welcome Message:** Founder (Pradeep) sends a personal WhatsApp welcome to the clinic owner.
- [ ] **Connection Check:** Verify if `whatsapp_connections` status is 'active'. If not, send a "Need help linking?" video.
- [ ] **Treatment Seeding:** Ensure the Onboarding Wizard has populated basic pricing.

## Day 1-3: The Data Phase
- [ ] **Patient Import:** Monitor if `patients` count > 0. 
- [ ] **The "Magic Moment":** Ensure the first automated reminder goes out. Once a clinician sees a patient respond via WhatsApp, they are 80% locked in.
- [ ] **Lead Engine Check:** If the clinic has active marketing, verify if leads are being ingested from Landing Pages.

## Day 7: The Insights Phase
- [ ] **The Weekly Brief:** Send a manual PDF or screenshots of the 'Reports' dashboard showing "No-Shows Prevented" and "Revenue Recovered".
- [ ] **Service Expansion:** Ask if they want to enable the 'Recall Engine' for lost patients.

## Day 14: Conversion
- [ ] **Feedback Call:** 10-minute call to discuss ROI.
- [ ] **Upgrade Flow:** Move them to the 'Standard' or 'Pro' plan based on patient volume.

---

### Red Flags (At-Risk Accounts)
1. **Zero WhatsApp activity in 48 hours:** Immediate outreach required.
2. **Disconnected WhatsApp session:** SMS alert to clinic admin.
3. **High "Blocked" message rate:** Audit the message guard logs to see if they are spamming unconsented patients.
